Hardware Requirements

This course involves hands-on development with Physical AI and Humanoid Robotics, which requires specific hardware for optimal performance.

Workstation Requirements​

A powerful workstation is essential for running simulations, training AI models, and developing complex robotics software.

ComponentRecommendationMinimum Specification
GPUNVIDIA RTX 3080 or better (12GB+ VRAM)NVIDIA RTX 3060 (8GB VRAM)
CPUIntel Core i7 (10th Gen+) / AMD Ryzen 7 (3000 series+)Intel Core i5 / AMD Ryzen 5
RAM32 GB DDR416 GB DDR4
Storage1 TB NVMe SSD500 GB SSD
Operating SystemUbuntu 20.04+ (Recommended for ROS 2 and NVIDIA SDKs)Windows 10/11 (with WSL2)

Edge Computing Kit​

For deploying AI models directly onto a robot or embedded system, an edge computing device is required.

DeviceRecommendationNotes
NVIDIA JetsonJetson Orin Nano Developer Kit (8GB/16GB)Ideal for AI inference on the edge
